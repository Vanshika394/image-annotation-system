## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Supabase account

### Installation

1. Clone the repository
```bash
git clone https://github.com/Vanshika394/image-annotation-system/edit/main
cd image-annotation-system
```

2. Install dependencies
```bash
npm install
```
```

4. Start the development server
```bash
npm run dev
```
